# Pest Development Index (PDI) for Agricultural Pest Outbreak Analysis

This repository provides the Python scripts used in the manuscript:

**Detection of Agricultural Pest Outbreaks from Antecedent Environmental Conditions Using a Satellite-Derived Pest Development Index**

The study proposes a satellite-based time-series framework for detecting antecedent environmental conditions associated with agricultural pest outbreaks in South Korea. The Pest Development Index (PDI) integrates temperature suitability, moisture conditions, short-term cumulative precipitation, and environmental persistence using meteorological observations and satellite-derived land surface variables.

## Overview

The analysis combines the following environmental variables:

- Air temperature from Korea Meteorological Administration (KMA)
- Relative humidity from KMA
- Daily precipitation from KMA
- Surface soil moisture from NASA SMAP Level 4
- Surface temperature from NASA SMAP Level 4

A total of 40 PDI combinations are generated by considering:

- Temperature source: air temperature and surface temperature
- Rainfall accumulation window: 1, 3, 5, and 7 days
- Persistence window: 1, 3, 7, 15, and 30 days

The performance of each PDI combination is evaluated using three pre-event metrics:

- Growth metric
- Persistence metric
- Peak timing metric

## Description of scripts

### timeseries.py

Generates time-series plots of meteorological and land surface conditions for each pest outbreak event. The plotted variables include air temperature, surface temperature, relative humidity, surface soil moisture, and daily precipitation.

### timeseries_PDI.py

Calculates the Pest Development Index (PDI) for each event. This script merges KMA and SMAP datasets, normalizes input variables, calculates PDI components, and generates 40 PDI time series based on different rainfall accumulation and persistence windows.

Main outputs include:

- PDI time-series Excel file
- Combined PDI CSV file
- PDI time-series plots
- Processing log file

### analysis1.py

Calculates the pre-event growth metric of the PDI. The growth metric evaluates how rapidly the PDI increases before the pest outbreak event.

Main outputs include:

- Growth metric summary CSV
- Growth score box plot
- Growth score heatmap

### analysis2.py

Calculates the persistence metric of the PDI. The persistence metric evaluates how long elevated PDI conditions are maintained during the pre-event period.

Main outputs include:

- Persistence metric summary CSV
- Persistence score box plot
- Persistence score heatmap

### analysis3.py

Calculates the peak timing metric of the PDI. The peak timing metric evaluates when the dominant pre-event PDI peak occurs relative to the pest outbreak date.

Main outputs include:

- Peak timing metric summary CSV
- Peak timing score box plot
- Peak timing score heatmap

## Required Python packages

The scripts require the following Python packages:

- numpy
- pandas
- matplotlib
- openpyxl

These packages can be installed using:

pip install numpy pandas matplotlib openpyxl

## Input data

The scripts require event-based KMA and SMAP time-series datasets. The expected input files include:

- kma_pests_daily_timeseries.xlsx
- smap_l4_surface_temp_sm_timeseries_1perday.xlsx

or event-based CSV files depending on the script.

Because the original meteorological and satellite datasets are obtained from external data providers, users should download or prepare the corresponding input data before running the scripts.

## How to run

Run the scripts in the following order:

python timeseries.py
python timeseries_PDI.py
python analysis1.py
python analysis2.py
python analysis3.py

Before running the scripts, please modify the BASE_DIR path in each Python file to match the local directory where the input data are stored.

## Outputs

The scripts generate:

- Time-series plots of meteorological and land surface variables
- PDI time-series files
- Growth, persistence, and peak timing metric summaries
- Box plots and heatmaps for comparing PDI combinations
